Detailed Description

Command Line Interface for the mojo-nes-mk3 programmer.

This utility allows to manage mojo-nes-mk3 cartridges, usina a mojo-nes-mk3 programmer. The utility allows to program and read flash and RAM chips. A driver system allows to support several mapper chip implementations.
